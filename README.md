# Principes de phonologie (N. Troubetzkoy) avec sons

## Description

Traditionnellement, les livres de phonétique et de phonologie, pour plus complets qu'ils soient dans leur description des phonèmes, ne permettent pas au lecteur ou à la lectrice de les entendre. L'objectif du présent projet est de permettre l'écoute des voyelles évoquées dans le deuxième (?) chapitre du livre *Principes de phonologie* (1949), de Nikolai Troubetzkoy, afin de mieux les saisir. Ainsi, toutes les voyelles présentes dans ce chapitre, et notées de façon standard d'après les conventions de l'API (Alphabet Phonétique International), sont associées à un bouton ; lorsque un clique est réalisé sur ce bouton, le phonème vocalique est reproduit.<br>
L'auteur n'a hélas pas noté toutes les voyelles d'après les conventions adoptées par l'API. Ainsi, dans les cas où la voyelle n'est pas notée de façon standard, un message indicant l'impossibilité de reproduire le phonème s'affiche lorsque la souris est positionnée sur la voyelle.
